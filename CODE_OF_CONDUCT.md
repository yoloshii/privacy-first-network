# Code of Conduct

## Our Standards

This project is focused on network privacy and security. We welcome contributions from anyone who shares that goal, regardless of background or experience level.

**Expected behavior:**
- Be respectful and constructive in discussions
- Focus feedback on the technical merits
- Help others learn — many users are deploying network infrastructure for the first time
- Report security issues responsibly (see [SECURITY.md](SECURITY.md))

**Unacceptable behavior:**
- Harassment, insults, or personal attacks
- Publishing others' private information
- Sharing VPN credentials, private keys, or other secrets in issues/PRs
- Promoting illegal activities (this is a privacy tool, not a hacking tool)

## Enforcement

Violations can be reported by opening an issue or contacting the maintainer. Reports will be reviewed and may result in temporary or permanent bans from the project.

## Scope

This code of conduct applies to all project spaces: issues, pull requests, discussions, and any public communication referencing this project.
